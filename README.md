A modern, interactive plant shop website built with React and Redux. Discover and purchase beautiful houseplants with an intuitive shopping experience.

## Features

- **Landing Page**: Welcoming introduction with background imagery and company information
- **Product Listing**: Browse 6 unique houseplants across 3 categories (Indoor, Outdoor, Succulents)
- **Shopping Cart**: Add, increase, decrease, or remove plants with real-time updates
- **Responsive Design**: Optimized for desktop and mobile devices
- **Interactive UI**: Hover effects, animations, and notifications for better user experience
- **State Management**: Redux for efficient cart and app state handling

## Technologies Used

- **React**: Component-based UI library
- **Redux**: State management
- **React Router**: Client-side routing
- **CSS3**: Custom styling with modern design principles
- **GitHub**: Version control and repository hosting

## Installation

1. Clone the repository:
   ```
   git clone https://github.com/PrakulSmarty/Plant-Paradise.git
   ```

2. Navigate to the project directory:
   ```
   cd plant-paradise
   ```

3. Install dependencies:
   ```
   npm install
   ```

4. Start the development server:
   ```
   npm start
   ```

5. Open your browser and visit `http://localhost:3000`

## Usage

- **Home**: View the landing page and click "Get Started" to browse products
- **Products**: Explore plants by category, add to cart (buttons disable after adding)
- **Cart**: Manage items with increase/decrease buttons, remove items, view total
- **Navigation**: Use the header to switch between pages
